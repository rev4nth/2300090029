
Notifications App Frontend

Summary

A simple frontend for notifications

Prerequisites

Node and npm installed

Steps to run

Clone the repository with git clone repository url
Open the project folder notifications_app_fe
Install dependencies npm install
Start the development server npm run dev
Open the app in the browser at localhost 5173 or the port shown by vite

Notes

Logs logic is in logs js
Api routing is in api js
Sorting and frontend are in notifications js
